1. Allgemein

Der PCM-APPServer dient zur Synchronisation zwischen der Desktopanwendung PCM-Manager und der PCM-APP. Die Synchronisation erfolgt verschlüsselt über SSL.

2. Systemvoraussetzungen

  Systemvoraussetzungen
  Betriebssystem:  Windows 8, Windows 10, Windows 11
  Prozessor:  Prozessor mit 1 GHz oder höher
  Arbeitsspeicher:  2 Gigabyte
  Festplattenspeicher:  ca. 1 Gigabyte
  Grafikkarte:  DirectX 9 oder höher
  Display:  1280 x 800
  Internetverbindung:  Für die Aufrufe über die REST-API
  Freigabe Ports:  55700 TCP ausgehend
  SQL-Server:  MYSQL 5.x für 32-Bit
  MYSQL 8.x für 64-Bit

3. Installation

Der PCM-Appserver wird automatisch über das Setup installiert, wenn dieser im Setup zur Installation ausgewählt wurde. Sollte der PCM-Appserver nachträglich installiert werden. Starten Sie die Kommandozeile als Administrator. Navigieren Sie ins Verzeichnis wo die PCMAPPServer.exe abgelegt ist und geben Sie PCMAPPServer.exe -install ein, um den Dienst zu installieren.
